# Frontend C3PO

Cette arborescence contient l'interface React/Vite/Tauri de C3PO.

La documentation complete du projet se trouve dans le README racine.

## Rappel rapide

- `npm run dev` lance Vite pour travailler sur l'UI.
- `npm run tauri dev` lance la version desktop, avec les plugins Tauri et le backend local.
- Le runtime desktop attend un backend compile dans `src-tauri/bin/start_backend/`.
